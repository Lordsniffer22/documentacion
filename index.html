Getting Started

To get started with the bot, add it to your Telegram contacts and start a conversation.

Commands

  • /start: Initiate a conversation with the bot.
  • /help: Get a list of available commands and their descriptions.

Extracting User Information

The bot can extract user information such as username and full name.

Extracting Username

username = message.from_user.username

Extracting Full Name

full_name = f"{message.from_user.first_name} {message.from_user.last_name or ''}".strip()
